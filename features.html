Features & Help

This page describes what Archery Assist can do today and how archers and coaches can talk to the system using WhatsApp.

1. Logging Training in Natural Language

Basic Logging

Archers can log arrows in plain English. Examples:

  • 120 arrows
  • 120 arrows @ 70m
  • Morning session 120 arrows @ 50m
  • Evening 100 arrows, windy

The system automatically extracts:

  • Arrow count
  • Distance (e.g., 70m, 18m)
  • Notes (e.g., windy, heavy rain, practice match)
  • Approximate session time (morning/evening, if mentioned)

2. Summary Commands

Daily / Weekly / Monthly Views

Archers and coaches can request summaries:

  • /today — Today’s arrow count.
  • /week — Arrow total for the last 7 days.
  • /month — Arrow total for the current calendar month.

These are useful for quickly checking if training volume is on track.

Recent Session History

/history — Shows a list of recent practice entries, including date, distance, and arrows shot.

3. Coach Tools (Beta)

Viewing Logs for an Archer

Coaches can query logs for a particular archer (format may evolve as the system matures):

  • /coach list <archer-name-or-id>

This shows all recent logs for that archer.

Date-Range Search

Coaches can request logs for a specific date range:

  • /coach search 2025-01-01 2025-01-20

This is helpful when reviewing preparation for a particular competition or camp.

Planned Competitions List

Coaches can maintain a list of recommended or target competitions for the season. This helps align training volume with the actual calendar.

4. Competition & Scoring Tracking

Competition Entries

Archers can log competition events and scoring rounds in a simple format:

  • competition: Lancaster 560
  • competition: State Indoor, 540

Scoring Rounds

Archers can also enter scoring-round data, for example:

  • scoring: 30 arrows @ 18m score 276

Coaches can review such rounds via coach-specific commands such as:

  • /coach competitions <archer>
  • /coach scores <archer>

5. Session Classification

Automatic Tagging

The system can classify sessions based on what the archer sends:

  • Time of day: morning / evening (if mentioned).
  • Indoor vs Outdoor: inferred from distance (e.g., 18m often indoor, 50m or 70m often outdoor).
  • Type of session: practice vs competition vs scoring round.

These tags are used to create more meaningful summaries over time.

6. Data Maintenance – Soft Delete

Correcting Recent Entries

Users can soft-delete recent entries if they made a mistake. For example:

  • /delete list — Show entries from the last 30 days with an ID.
  • /delete 5 — Delete the item with ID 5 from that list.

Soft delete makes corrections possible without exposing older data or complex database controls to the end user.

7. Quick Help Command

/help

Sending /help in WhatsApp will show a brief list of available commands, similar to:

Available Commands:
 /today  – Today’s arrow count
 /week   – Weekly arrow summary
 /month  – Monthly arrow summary
 /history – List recent sessions
 /delete – Manage recent logs
 /coach  – Coach-only commands (if applicable)
